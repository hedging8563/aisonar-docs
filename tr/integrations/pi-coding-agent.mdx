## Genel Bakış

<Note>
**Tür**: Kodlama aracı

**Birincil yol**: OpenAI ile uyumlu yol

**Destek seviyesi**: Desteklenen yol
</Note>

[Pi](https://github.com/badlogic/pi-mono/tree/main/packages/coding-agent) minimal bir terminal kodlama aracıdır. TokenLab dokümantasyon IA'sında yeri **Kodlama araçları** grubudur; native SDK değildir.

Pi şu kullanım tarzlarının arasında durur:

- etkileşimli terminal kodlama iş akışı
- süreç entegrasyonu için JSON / RPC modları
- kendi uygulamalarınıza gömülebilen SDK tarzı çalışma ortamı

TokenLab ile en basit başlangıç noktası genelde **OpenAI ile uyumlu** yoludur; Anthropic-native veya Google-native davranışı bilerek istemiyorsanız buradan başlayın.

## Önerilen yol

```bash
export OPENAI_API_KEY="sk-your-tokenlab-key"
export OPENAI_BASE_URL="https://api.tokenlab.sh/v1"
```

Ardından Pi'nin sağlayıcı veya özel model yolunu TokenLab'ya yönlendirin.

<Note>
Claude-native veya Gemini-native davranış gerekiyorsa, bunu genel OpenAI ile uyumlu ayar olarak değil, sağlayıcıya özel Pi yapılandırması olarak ele alın.
</Note>
