## Önerilen akış

1. [Modelleri listele](/tr/api-reference/models/list-models) çağrısını `recommended_for=3d` ile yapın.
2. Metin `prompt` ile [3D oluştur](/tr/api-reference/3d/create-3d) kullanın; `image` veya `image_url` yalnızca image-to-3D destekleyen modellerde eklenmelidir.
3. `format`, `quality`, `style` veya `seed` gibi isteğe bağlı alanları seçilen model sözleşmesine göre seçin.
4. Varlık hazır olana kadar `poll_url` yoklayın, sonra `model_url` veya `glb_url` / `fbx_url` gibi format URL’lerini indirin.

## Modelleri keşfet

3D üretimi görev tabanlıdır. Tek bir sağlayıcı veya sabit model varsaymak yerine model kataloğunu doğruluk kaynağı kabul edin.

```bash
curl "https://api.aisonar.dev/v1/models?recommended_for=3d" \
  -H "Authorization: Bearer sk-your-api-key"
```

## 3D görevi oluştur

Text-to-3D istekleri `prompt` gerektirir. Image-to-3D istekleri base64 `image` veya `image_url` ekler. `format` ve `quality` yalnızca model sözleşmesi sunuyorsa kullanılmalıdır.

```bash
curl https://api.aisonar.dev/v1/3d/generations \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tripo3d-v2.5",
    "prompt": "A stylized low-poly robot mascot with clean topology",
    "format": "glb",
    "quality": "standard"
  }'
```

## Sonuçları yokla

Önce dönen `poll_url` değerini kullanın. Kararlı fallback [Görev durumu](/tr/api-reference/tasks/get-task-status) endpointidir. Tamamlanan 3D görevleri `model_url` döndürür ve `glb_url` veya `fbx_url` içerebilir.

```bash
curl "https://api.aisonar.dev/v1/tasks/$TASK_ID" \
  -H "Authorization: Bearer sk-your-api-key"
```
