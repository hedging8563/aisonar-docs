## Önerilen akış

1. [Modelleri listele](/tr/api-reference/models/list-models) çağrısını `recommended_for=music` ile yapın.
2. [Müzik oluştur](/tr/api-reference/music/create-music) ile `prompt` ve isteğe bağlı `title`, `tags`, `action`, `mv` veya devam ayarlarını içeren bir görev oluşturun.
3. Dönen `id` / `task_id` ve `poll_url` değerlerini saklayın.
4. Görev `completed` veya `failed` olana kadar yoklayın, ardından `audio_url`, isteğe bağlı `video_url`, `title` ve `lyrics` alanlarını okuyun.

## Modelleri keşfet

Müzik üretimi asenkrondur. Upstream sağlayıcı adına veya sağlayıcı görev ID’sine bağlı geliştirme yapmayın; AI Sonar normalize edilmiş görev kimliği ve polling URL’i döndürür.

```bash
curl "https://api.aisonar.dev/v1/models?recommended_for=music" \
  -H "Authorization: Bearer sk-your-api-key"
```

## Müzik görevi oluştur

`action: "MUSIC"` prompt ve modele göre şarkı veya enstrümantal üretir. `action: "LYRICS"`, seçilen model desteklediğinde sadece söz üretimi içindir. `continue_clip_id` ve `continue_at` yalnızca model sözleşmesinde belgelenen devam akışlarında kullanılmalıdır.

```bash
curl https://api.aisonar.dev/v1/music/generations \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "suno_music",
    "prompt": "An upbeat synth-pop track with warm vocals and a clean chorus",
    "title": "Morning Static",
    "tags": "synth-pop, upbeat",
    "action": "MUSIC"
  }'
```

## Sonuçları yokla

Her zaman dönen `poll_url` değerini tercih edin. Kararlı fallback [Görev durumu](/tr/api-reference/tasks/get-task-status) endpointidir. Tamamlanan müzik görevleri `audio_url`, `video_url`, `title`, `lyrics` ve model tarafından normalize edilmiş metadata içerebilir.

```bash
curl "https://api.aisonar.dev/v1/tasks/$TASK_ID" \
  -H "Authorization: Bearer sk-your-api-key"
```
