## التدفق الموصى به

1. استدعِ [قائمة النماذج](/ar/api-reference/models/list-models) مع `recommended_for=image`.
2. افتح [جلب النموذج](/ar/api-reference/models/get-model) للنموذج المختار واتبع عقده العام.
3. استخدم [إنشاء صورة](/ar/api-reference/images/create-image) لتحويل النص إلى صورة وتدفقات الصور المرجعية المدعومة، أو [تحرير صورة](/ar/api-reference/images/edit-image) لتدفقات التحرير.
4. إذا احتوى الرد على `poll_url`، فاستطلع هذا الرابط حتى تصل المهمة إلى حالة نهائية.

## اكتشاف النماذج

لا تتشابه نماذج الصور في شكل الطلب. بعضها يعيد نتيجة مباشرة، وبعضها ينشئ مهمة غير متزامنة. يحدد عقد النموذج المختار ما إذا كانت الحقول مثل `size` و `quality` و `aspect_ratio` و `resolution` و `async` مقبولة.

```bash
curl "https://api.aisonar.dev/v1/models?recommended_for=image" \
  -H "Authorization: Bearer sk-your-api-key"
```

## نص إلى صورة

في نماذج عائلة GPT Image، استخدم `size` عندما يتيحه عقد النموذج. في نماذج Google أو Nano Banana، استخدم `aspect_ratio` و `resolution` فقط عندما يذكرهما عقد النموذج.

```bash
curl https://api.aisonar.dev/v1/images/generations \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-image-2",
    "prompt": "A clean product photo of a ceramic coffee cup on a walnut desk",
    "size": "1024x1024",
    "response_format": "url"
  }'
```

## الصور المرجعية والتحرير

لإنشاء صورة مرجعية على `/v1/images/generations`، أرسل `operation: "image-to-image"` مع `image_url` أو `image_urls` أو `reference_image_urls` أو ملف multipart باسم `image` عندما يدعمه النموذج. استخدم `/v1/images/edits` لنماذج التحرير التي تقبل صور multipart أو روابط الصور أو `images[].image_url` / `images[].file_id`.

لا ترسل `images[]` أو `file_id` في المستوى الأعلى إلى `/v1/images/generations`؛ هذه الأشكال تخص تدفقات التحرير التي توثقها صراحة.

```bash
curl https://api.aisonar.dev/v1/images/generations \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana-pro",
    "operation": "image-to-image",
    "prompt": "Keep the same product, place it in a bright studio scene",
    "image_urls": ["https://example.com/reference.png"],
    "aspect_ratio": "1:1"
  }'
```

## استطلاع النتائج

يرجع رد الصورة المتزامن `data[].url` أو `data[].b64_json`. ويرجع رد المهمة `status` و `task_id` وغالباً `poll_url`. فضّل `poll_url`؛ ونقطة fallback الثابتة هي [حالة المهمة](/ar/api-reference/tasks/get-task-status). مهام الصور غير المتزامنة تعيد روابط URL، لذلك اطلب base64 inline فقط في التدفقات المتزامنة التي تدعمه.

```bash
curl "https://api.aisonar.dev/v1/tasks/$TASK_ID" \
  -H "Authorization: Bearer sk-your-api-key"
```
