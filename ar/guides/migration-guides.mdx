## نظرة عامة

يوضح هذا الدليل كيفية استخدام أدلة الترحيل في الإنتاج دون ربط التطبيق بتفاصيل تنفيذ المزوّد.

اعتبر واجهات AI Sonar العامة وبيانات النموذج وحقول الاستجابة الموثقة هي العقد.

## السلوك الحالي

- راجع هذه القيم قبل الإصدار: base URL, API key, model ID.
- لا تعتمد على URL حالة خاص بالمزوّد أو معرّفات توجيه أو بيانات تنفيذ خاصة.
- اجعل retries آمنة للتكرار وتجنب إنشاء عمل قابل للفوترة مرتين.

## قائمة فحص الإنتاج

- يجب أن تساعد السجلات الدعم دون تخزين أسرار أو ملفات خاصة أو prompts أو وسائط مستخدم.
- يجب أن تتضمن تذاكر الدعم شكل الطلب بعد التنظيف، endpoint، النموذج، الوقت، والمعرّفات العامة.
- تحقق من المسار في staging قبل نقل حركة الإنتاج.
