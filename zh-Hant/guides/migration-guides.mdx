## 概述

本指南說明如何在生產環境中使用遷移指南，同時避免把應用綁定到供應商實作細節。

以 AI Sonar 公開 API、模型中繼資料和已文件化回應欄位作為契約。

## 目前行為

- 上線前重點檢查這些值：base URL, API key, model ID。
- 不要依賴供應商專有狀態 URL、路由識別符或私有執行中繼資料。
- 讓重試保持冪等，避免建立重複計費工作。

## 生產檢查清單

- 日誌要便於支援排障，但不要儲存密鑰、私有檔案、prompt 或使用者媒體。
- 支援工單應包含脫敏請求結構、端點、模型、時間戳和公開識別符。
- 生產流量切換前先在預發環境驗證流程。
