---
title: "Criar Modelo 3D"
openapi: "POST /v1/3d/generations"
description: "Cria uma tarefa de geração de modelo 3D"
---

Gere modelos 3D a partir de texto ou imagens usando Tripo3D e outros provedores. Esta é uma API assíncrona. A resposta de criação retorna uma identidade de tarefa e, quando disponível, também pode incluir um `poll_url` preferido.

As operações públicas de 3D são inferidas pelo formato da requisição: requisições somente com `prompt` resolvem para `text-to-3d`, e requisições com `image` ou `image_url` resolvem para `image-to-3d`. Este endpoint não expõe um campo de requisição `operation`; não envie o valor legacy `3d-generation`.

## Corpo da Requisição

<ParamField body="model" type="string" default="tripo-h3.1">
  Modelo a ser usado (ex: `tripo-h3.1`).
</ParamField>

<ParamField body="prompt" type="string" required>
  Descrição em texto do modelo 3D a ser gerado.
</ParamField>

<ParamField body="image" type="string">
  Imagem codificada em Base64 para geração de imagem para 3D.
</ParamField>

<ParamField body="image_url" type="string">
  URL da imagem para geração de imagem para 3D.
</ParamField>

<ParamField body="format" type="string" default="glb">
  Formato de saída: `glb`, `fbx`, `obj` ou `usdz`.
</ParamField>

<ParamField body="quality" type="string" default="standard">
  Nível de qualidade: `draft`, `standard` ou `high`.
</ParamField>

<ParamField body="style" type="string">
  Predefinição de estilo para o modelo.
</ParamField>

<ParamField body="seed" type="integer">
  Seed para geração reproduzível.
</ParamField>

<ParamField body="user" type="string">
  Um identificador exclusivo para o usuário final.
</ParamField>

## Resposta

<ResponseField name="id" type="string">
  ID da tarefa para consulta de status (polling).
</ResponseField>

<ResponseField name="task_id" type="string">
  Alias do identificador da tarefa assíncrona quando retornado pelo adaptador.
</ResponseField>

<ResponseField name="poll_url" type="string">
  URL de polling prioritária quando disponível.
</ResponseField>

<ResponseField name="status" type="string">
  Status da tarefa: `pending`, `processing`, `completed` ou `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  Timestamp Unix da criação da tarefa.
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":   "tripo-h3.1",
        "prompt":  "A detailed medieval castle with towers",
        "format":  "glb",
        "quality": "high",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.aisonar.dev/v1/3d/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```
