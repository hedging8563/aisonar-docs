## Ringkasan

Google Gemini CLI adalah alat baris perintah (command-line tool) untuk berinteraksi dengan model Gemini. TokenLab menyediakan endpoint yang kompatibel yang memungkinkan Anda menggunakan Gemini CLI dengan akses ke ratusan model.


<Warning>
Anggap integrasi ini sebagai eksperimental: Gemini CLI saat ini belum mendokumentasikan alur Base URL kustom ala TokenLab secara stabil. Jika Anda membutuhkan integrasi TokenLab terminal yang benar-benar didukung, gunakan Codex CLI, Claude Code, atau OpenCode.
</Warning>


<Note>
**Jenis**: Alat coding

**Jalur utama**: Gemini-native

**Tingkat dukungan**: Upaya terbaik
</Note>

## Persyaratan Sistem

- **Node.js**: Versi 20.0+
- **OS**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+, atau Debian 10+

## Instalasi

```bash
npm install -g @google/gemini-cli
```

Verifikasi instalasi:

```bash
gemini --version
```

## Konfigurasi

### Langkah 1: Dapatkan API Key Anda

1. Masuk ke [TokenLab Dashboard](https://tokenlab.sh/dashboard)
2. Navigasi ke [API Keys](https://tokenlab.sh/dashboard/api)
3. Buat dan salin API key Anda (format: `sk-...`)

### Langkah 2: Atur Environment Variables

**Sementara (sesi saat ini):**

```bash
export GEMINI_API_KEY="sk-your-tokenlab-key"
export GOOGLE_GEMINI_BASE_URL="https://api.tokenlab.sh"
```

**Konfigurasi permanen:**

Tambahkan ke file konfigurasi shell Anda:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-tokenlab-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.tokenlab.sh"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-tokenlab-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.tokenlab.sh"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-tokenlab-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.tokenlab.sh"
    ```
  </Tab>
</Tabs>

<Warning>
Mulai ulang terminal Anda setelah konfigurasi agar perubahan dapat diterapkan.
</Warning>

<Note>
**Batasan yang diketahui**: Dalam beberapa kasus, Gemini CLI mungkin tidak mematuhi `GOOGLE_GEMINI_BASE_URL` jika terdapat sesi Google yang tersimpan di cache. Jika Anda mengalami masalah koneksi, coba hapus cache CLI atau mulai sesi terminal baru.
</Note>
