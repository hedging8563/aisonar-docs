## Ikhtisar

Panduan ini menjelaskan cara memakai Keamanan dan penanganan data di produksi tanpa mengikat aplikasi ke detail implementasi provider.

Gunakan API publik AI Sonar, metadata model, dan field respons terdokumentasi sebagai kontrak.

## Perilaku saat ini

- Periksa nilai berikut sebelum rilis: management token, API key, media URL.
- Jangan bergantung pada URL status khusus provider, identifier routing, atau metadata eksekusi privat.
- Buat retry idempotent dan hindari membuat pekerjaan billable ganda.

## Checklist produksi

- Log harus membantu support tanpa menyimpan secret, file privat, prompt, atau media pengguna.
- Ticket support sebaiknya menyertakan request shape yang sudah disanitasi, endpoint, model, timestamp, dan identifier publik.
- Verifikasi flow di staging sebelum memindahkan traffic produksi.
