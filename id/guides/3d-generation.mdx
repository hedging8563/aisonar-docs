## Alur yang disarankan

1. Panggil [Daftar model](/id/api-reference/models/list-models) dengan `recommended_for=3d`.
2. Gunakan [Buat 3D](/id/api-reference/3d/create-3d) dengan `prompt` teks; tambahkan `image` atau `image_url` hanya untuk model yang mendukung gambar-ke-3D.
3. Pilih field opsional seperti `format`, `quality`, `style`, atau `seed` sesuai kontrak model.
4. Polling `poll_url` sampai aset siap, lalu unduh `model_url` atau URL khusus format seperti `glb_url` / `fbx_url`.

## Temukan model

Generasi 3D berbasis tugas. Jadikan katalog model sebagai sumber kebenaran, bukan mengasumsikan satu provider atau satu model tetap.

```bash
curl "https://api.aisonar.dev/v1/models?recommended_for=3d" \
  -H "Authorization: Bearer sk-your-api-key"
```

## Buat tugas 3D

Request teks-ke-3D memerlukan `prompt`. Request gambar-ke-3D menambahkan data base64 `image` atau `image_url`. Gunakan `format` dan `quality` hanya jika tersedia di kontrak model.

```bash
curl https://api.aisonar.dev/v1/3d/generations \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tripo3d-v2.5",
    "prompt": "A stylized low-poly robot mascot with clean topology",
    "format": "glb",
    "quality": "standard"
  }'
```

## Polling hasil

Gunakan `poll_url` yang dikembalikan terlebih dahulu. Fallback stabilnya adalah [Status tugas](/id/api-reference/tasks/get-task-status). Tugas 3D selesai mengembalikan `model_url` dan dapat menyertakan `glb_url` atau `fbx_url`.

```bash
curl "https://api.aisonar.dev/v1/tasks/$TASK_ID" \
  -H "Authorization: Bearer sk-your-api-key"
```
